# test
Rick this project was generated using the following vue cli steps

```
  1. Vue create test

  2. Manually select features
	   - babel
	   - router
	   - vex
	   - css pre-processors
	   - linter
	   - unit testing

   3. Use history mode for router = yes
   
   4. Sass/Scss (with node-sass)

   5. ESLint + Airbnb config

   6. Lint on save

   7. Mocha + chai

   8. Dedicated config files

   9. Yarn
```

## Project setup

yarn install
